
# Fashion-hub

Fashion-hub is an ecommerce store for the latest clothes .



## Features

### Home Page 
###  Product listing page
###  Authentication page
###  Filtering and Sorting
###  Cart Management
- ADD
- REMOVE
- Increase/Decrease Quantity
- ADD to Wishlist
### Wishlist Management
- ADD
- ADD to CART
- REMOVE


## Tech Stack
Client: React, React Router, Context, Reducer

Server: Mockbee (Mock Backend)
